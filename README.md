# Clone
```
git clone https://github.com/saracalihan/mavidurak-bot.git
cd mavidurak-bot
```
# Install
```
npm install
```
# Rename Env File
Rename `example.env` file then change bot token
```
mv example.env .env
```
# Run
```
npm start
```
# Bot Commands
`sudo help`: bot send all comands to you with DM.
